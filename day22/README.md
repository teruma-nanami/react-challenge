# day22 学習内容

本日の学習内容は以下の通りです。

- React Router の基礎
- React Router を用いた SPA の構築
- ルーティングの設定方法
- セッションストレージを用いたログイン機能の実装

## メモアプリ作成

本日は React Router を使用して、シンプルなメモアプリを作成しました。以下に主要なポイントをまとめます。

### 1. React Router のインストール

まず、React Router をプロジェクトにインストールしました。

```bash
npm install react-router-dom
```

### 2. ルーティングの設定

`BrowserRouter`、`Routes`、`Route`コンポーネントを使用して、ルーティングを設定しました。

### 3. コンポーネントの作成

- `Home`コンポーネント: メモの一覧を表示
- `NewMemo`コンポーネント: 新しいメモを追加するフォーム
- `EditMemo`コンポーネント: 既存のメモを編集
- `About`コンポーネント: アプリの説明

### 4. ナビゲーションの実装

`Link`コンポーネントを使用して、ページ間のナビゲーションを実装しました。

### 5. メモの管理

- ブラウザのローカルストレージを使用してメモを保存・取得
- メモの追加、編集、削除機能を実装

### 6. ログイン機能の追加実装

- sessionStorage を使用して簡易的なログイン機能を実装しました。
- ログイン状態に応じて表示内容を変更
- ログアウト機能の実装
- ログインページの作成
- ログイン状態の管理については実装できてないので次回実装予定

### 7. スタイリング

- Chakra UI を使用して、アプリの見た目を整えました。

### 8. 動作確認

- 各ページの動作確認を行い、メモの追加、編集、削除が正常に動作することを確認しました。
- ログインが正しく機能することを確認しました。
- 本番と同様に registeredUser と user を sessionStorage に保存して責務の分離を行いました。
