# React 就職チャレンジ

このチャレンジはプログラミングスクールを卒業したのちに、なかなか就職することが難しい人が作成した 150 日間の学習チャレンジである。

## ルール

- 就職が決まり次第終了
- 期間は 150 日間で 1000 時間の学習を目指す
- 毎日何らかのサイト/アプリをデプロイ
- デプロイは自分のサイトで行う
- 学習コードは GitHub で公開
